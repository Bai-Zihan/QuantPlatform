# 量化股票分析平台
### by ziyi020924-png
这是由用户VaskaLee开发的本地运行的股票走势分析雏形，支持从公开网络数据源读取行情，计算常见技术指标，并做简单的均线交叉回测。由于作者出身于非技术背景，交予Bai-Zihan进行项目管理和辅助开发。

## 功能

- A 股日线行情读取，默认使用 `akshare`
- K 线图、成交量、均线、布林带
- RSI、MACD 技术指标
- 简单双均线交叉策略回测
- 支持上传同花顺等平台导出的 CSV 文件作为本地数据源

## 安装

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

## 桌面 App 运行

双击项目里的 `QuantPlatform.app` 即可打开原生桌面版。这个版本基于 PySide6/Qt，不会在后台启动 Streamlit，也不会自动打开浏览器。

当前根目录的 `QuantPlatform.app` 是 PyInstaller 打包后的完整应用包，不再依赖项目里的 `.venv`。

如果 macOS 提示无法打开，可以在终端运行：

```bash
open QuantPlatform.app
```

如果仍然打不开，可以运行：

```bash
QuantPlatform.app/Contents/MacOS/QuantPlatform
```

把终端里显示的错误发出来继续排查。

桌面版会输出量化信号，包括：

- 股票名称和股票代码
- 金融科技深色界面和自定义 App 图标
- 图表图例，标明收盘价、MA20、MA60、RSI、DIF、DEA、买入持有、均线策略等线条
- 买入、加仓、观望、减仓、卖出
- 建议仓位
- 综合评分和置信度
- 风险等级
- 触发原因和风险提示
- “方法”页解释每个指标为什么用于分析
- “量化指标”页显示行业常用风险收益公式：CAGR、年化波动率、Sharpe、Sortino、Calmar、VaR、CVaR、ATR、Kelly 上限仓位、波动率目标仓位

这些信号仅用于量化研究，不构成投资建议。

## 股票代码格式

- 网络行情：可以输入股票名称、简称关键词或 6 位代码，例如 `贵州茅台`、`茅台`、`600519`、`000001`、`300750`
- 本地 CSV：只是备用导入方式，不是必需

网络行情会先解析名称/代码，再读取行情。名称搜索默认使用腾讯股票搜索接口，行情默认使用 AkShare 的腾讯历史行情接口，失败时会继续尝试东方财富历史行情接口。

## CSV 字段兼容

平台导出的字段名可能不一致。本项目会自动识别常见字段：

- 日期：`date`、`日期`、`交易日期`
- 开盘：`open`、`开盘`、`开盘价`
- 最高：`high`、`最高`、`最高价`
- 最低：`low`、`最低`、`最低价`
- 收盘：`close`、`收盘`、`收盘价`
- 成交量：`volume`、`成交量`

## 说明

同花顺通常没有面向个人开发者的稳定开放行情 API。更稳妥的方式是：

1. 使用同花顺导出的 CSV 做离线分析；
2. 使用 AkShare、Tushare、交易所公开数据或券商官方 API 做在线数据；
3. 后续如果有合法授权的数据接口，可以只替换 `data_provider.py` 中的数据读取逻辑。

本项目仅用于学习和研究，不构成投资建议。
